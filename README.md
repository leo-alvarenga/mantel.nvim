# mantel.nvim

A dead-simple, customizable and cozy buffer/tabline.

## Preview

...

## Features

- **No configuration required to get started**
- Easy setup and configuration
- No dependencies (unless you want to use the optional icons feature, which requires a Nerd Font)

## Installation

### Using [vim-plug](https://github.com/junegunn/vim-plug):

```vim
Plug 'leo-alvarenga/mantel.nvim'
```

### Using [lazy.nvim](https://github.com/folke/lazy.nvim):

```lua
{
  'leo-alvarenga/mantel.nvim',
  branch = 'nightly', -- Optional: specify the nightly branch if you want to use the latest features
  opts = {}, -- No configuration is required to get started
}
```
